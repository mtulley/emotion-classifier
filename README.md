# Emotion Classifier
Fine-tuned BERT for multilabel emotion classification. Trained on a dataset from HuggingFace: ttxy/emotion. Classifies text into 6 classes: anger, fear, suprise, love, joy, sadness. 
## Files
- fine-tune-BERT.ipynb: Jupyter Notebook which trains and evaluates BERT for emotion classification. Handles dataset downloading, processing, training, and evaluation.
- requirements.txt: Run ```pip install -r requirements.txt``` to install all dependencies. 
## How to run
Recommended to run in a colab session. Dependencies contained in requirements.txt. Install dependencies then run fine-tune-BERT.ipynb.
